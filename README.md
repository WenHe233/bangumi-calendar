# 新番时间表

按星期 / 时段排版的本季 & 历季新番时间表。纯前端、单文件，支持 PWA。

## 功能

- 季度索引：选年份 + 季度（1月/4月/7月/10月），从 [bangumi-data](https://github.com/bangumi-data/bangumi-data) 自动加载番剧
- 当前热播：显示最近若干天内开播的在播番剧
- 按星期分组、按时段排序，自带「NOW」分割线标记当前时间
- 30 小时制 / 保留当天 0-6 点动画（凌晨档归入前一天）
- 多类型筛选（TV / Web / 剧场版 / OVA）
- 标题语言切换：简中 / 繁中 / English / 日本語
- 一键跳转 Bangumi / Bilibili / 爱奇艺 / 腾讯视频 / 巴哈姆特动画疯 / 木棉花 / Ani-One 等播放源
- PWA 离线可用，可「添加到主屏幕」
- 移动端筛选抽屉

## 使用

直接在浏览器里打开 `index.html` 即可，也可以丢到 GitHub Pages / 任意静态服务器上。
